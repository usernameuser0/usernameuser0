Redis的下载安装

2023/8/13 Redis

1. 下载安装

1.1 Redis文件

去虚拟机安装之前先安装gcc

Yum install gcc 
安装gcc

1.2 然后解压压缩包

Tar -zxvf 文件名

1.3 进入解压文件执行编译

Make

1.4 编译之后安装

make Install

2. 后台启动

2.1配置文件备份

redis.conf

后台启动设置daemonize no改成yes

redis-server /home/redis/redis.conf

启动时指定文件

requirepass
密码设置

3. redis关闭
